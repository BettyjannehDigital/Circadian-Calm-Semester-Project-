# Circadian Calm – Semester Project

**Semester Project | Noroff Digital Marketing | March 2026**  
**Grade: C (D on first submission, improved)**

---

## Overview

Circadian Calm is a fictional educational wellness brand created for the Noroff Digital Marketing semester project. The brief required building a live content hub targeting a specific audience, demonstrating strategy, content creation, SEO, usability, and accessibility.

The concept was inspired by real professional experience — as a night shift care assistant, I built a resource I genuinely wished existed: structured, research-informed sleep and recovery guidance for shift workers.

The site was built on WordPress and hosted at: [oneohonejewelry.com/circadian-calm](https://www.oneohonejewelry.com/circadian-calm/)
